================
Doodle for TYPO3
================

This extension integrates Doodle (http://doodle.com) within a TYPO3 website.


Installation
============

::

    $ cd /path/to/website/typo3conf/ext/
    $ git clone https://github.com/xperseguers/t3ext-doodle.git doodle
    $ cd doodle/Classes/
    $ composer install
